Widget Integration
 Follow the below steps to integrate chat widget in your code.
Usage in HTML

First, place the below code inbetween <head> <head/> tags 

                          <script type="text/javascript" src="https://sample-app-pj5x.onrender.com/vf-chat-widget.js"></script>
                              <link rel="stylesheet" href="https://sample-app-pj5x.onrender.com/styles.css">

Add <vf-chat-widget><vf-chat-widget/> in your code. 

                          
                              <vf-chat-widget
                                  account-id="{{config.accountId}}"
                                  title="{{config.title}}"
                                  greeting="{{config.greeting}}"
                                  placeHolder="{{config.placeholder}}">
                              </vf-chat-widget>
Usage in Angular 2+

First, Install vf-chat-widget package from npm

npm install --save vf-chat-widget

Import chat widget into your module app.module.ts 

import { VfChatWidgetModule } from 'vf-chat-widget';
imports: [ VfChatWidgetModule ]

Add <vf-chat-widget><vf-chat-widget/> in your component. 

                          
                              <vf-chat-widget
                                  accountId="{{config.accountId}}"
                                  title="{{config.title}}"
                                  greeting="{{config.greeting}}"
                                  placeHolder="{{config.placeholder}}">
                              </vf-chat-widget>
